# LastFm-Extrator-Semanal
Script em Python para extrair automaticamente artistas escutados na última semana no Last.fm e gerar relatórios em CSV e logs em JSON.

Configuração de ambiente: rode ```pip install -r requirements.txt```

Você consegue testar o sistema inserindo sua chave da API em `https://www.last.fm/api/account/create`.

Terá como retorno 1 arquivo em CSV (registra semanal do usuário desejado) e 1 arquivo em JSON (log)
